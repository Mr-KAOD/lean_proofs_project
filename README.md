About
-
This repository contains the files used while researching the formalizing of mathematics using Lean, as undergraduate research assistants for Dr. Jun Liu, PhD. The project focuses on translating mathematical concepts, definitions, and proofs from natural language into a formal language that can be verified by a computer. This work combined mathematical proof analysis, formal verification, and functional programming.

Current Progress (since 27 May 2026)
-
Formalizing basic mathematical results, such as  
- the sum of natural numbers, 1 + 2 + 3 + ... + n = n(n+1)/2
- Cassini's Identity for Fibonacci Numbers, F(n)F(n+2) - F(n+1)^2 = (-1)^(n+1)
